## Our commitment

This project has adopted the [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/). All participants — contributors, maintainers, and community members — are expected to uphold these standards in every interaction related to this project.

## Principles

The Microsoft Open Source Code of Conduct asks everyone to:

- **Be respectful** — treat all contributors and community members with courtesy and professionalism
- **Be inclusive** — welcome people of all backgrounds, experience levels, and perspectives
- **Be constructive** — focus feedback on ideas and code, not on individuals
- **Be collaborative** — work together openly and assume good intent

<Info>
  These principles apply across all project spaces: GitHub issues, pull requests, code review comments, and any other forums associated with the PlayFab Samples repository.
</Info>

## Unacceptable behavior

The following are examples of behavior that violates this code of conduct:

- Harassment, threats, or personal attacks of any kind
- Discriminatory language or jokes related to race, gender, sexual orientation, disability, religion, or similar characteristics
- Publishing others' private information without explicit permission
- Any other conduct that would be considered inappropriate in a professional setting

## Reporting violations

If you experience or witness behavior that violates the code of conduct, report it by contacting the Microsoft Open Source team:

**Email:** [opencode@microsoft.com](mailto:opencode@microsoft.com)

<Note>
  All reports will be reviewed and investigated. The project team is committed to maintaining confidentiality for anyone who reports an incident.
</Note>
